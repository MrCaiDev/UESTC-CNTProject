# 计算机网络与通信技术

电子科技大学 2020 级《计算机网络与通信技术》课程代码。

## ✨ 项目介绍

1. 最小网元设计（Socket）
2. 校园互联网（Packet Tracer）
3. 网络流量抓包分析（WireShark）
4. 数字数据调制解调（Simulink）
5. 各单元作业
    - 词汇解释
    - 教材习题
    - 思维导图
    - 课堂讨论
